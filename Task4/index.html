Pipeline Summary

  • Infrastructure created by Terraform: VPC networking (via AWS defaults), Security Group (22/80), and EC2 with user-data bootstrapping Python.
  • Configuration applied by Ansible: install & enable NGINX, deploy this HTML, ensure idempotency.
  • Speed: zero-click provisioning after $ terraform apply → $ ansible-playbook.

Re-run locally

# 1) Provision infra
terraform init && terraform apply -auto-approve

# 2) Update inventory with EC2 IP
echo "[web]\n‹EC2_IP› ansible_user=ec2-user ansible_ssh_private_key_file=~/.ssh/terraform-key" > inventory

# 3) Configure with Ansible
ansible-playbook -i inventory site.yml

# 4) Test
open http://‹EC2_IP› # or xdg-open

Tip: remember terraform destroy to avoid charges.
